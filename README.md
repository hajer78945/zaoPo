### 一、中餐宴会的基本礼仪：
#### 1. 桌次排列
在中餐宴请活动中，往往采用圆桌布置菜肴、酒水。排列圆桌的尊卑次序，有两种情况。

第一种情况，是由两桌组成的小型宴请。这种情况，又可以分为两桌横排和两来竖排的形式。

当两桌横排时，桌次是以左为尊，以右为卑。这里所说的右和左，是由面对正门的位置来确定的。
